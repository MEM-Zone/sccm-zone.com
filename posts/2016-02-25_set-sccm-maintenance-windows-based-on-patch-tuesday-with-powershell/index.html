The script is using a ‚Äúconfiguration‚Äù CSV where you can specify the script parameters.

  • CollectionName Specifies the collection name for which to set maintenance windows.
  • Year Specifies the maintenance window year.
  • Month
    Specifies the maintenance window month.
  • OffsetWeeks Specifies the maintenance window offset number of weeks after patch Tuesday.
  • OffsetDays Specifies the maintenance window offset number of days after path Tuesday.
  • StartTime Specifies the maintenance window start time.
  • StopTime Specifies the maintenance window stop time.
  • SetForWholeYear Specifies if to set the maintenance windows for the whole year.
  • RemoveExisting Specifies if to remove existing maintenance windows.
  • ApplyTo Specifies the maintenance window type ( Any | SoftwareUpdates | TaskSequences).
